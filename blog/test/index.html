Hugo 操作

想要把自己的個人網站發布到 Github Pages 的話，要在 cmd 上輸入 hugo，這樣就會產生出一個 public 的資料夾

再把這資料夾裡的東西複製到 C:\Users\XXX\Documents\GitHub\ 你的 repo ，然後再上傳到 github 就行了

之後就能在網址上輸入 你的 github 帳號.github.io

如何看自己選的 Hugo 主題有沒有辦法使用評論功能

在 config.toml 底下去找有沒有叫 disqusShortname 的東西，如果有就代表可以使用
