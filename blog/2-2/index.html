Spring Boot IoC 2-2 重點

By John 21 November 2022

2-2 章

@component
  1. 用法:只能加在 class 上
  2. 用途:將該 class 變成由 Spring 容器所管理的 object ，那這些 object 會被稱作為 bean(這個 bean 名字為 class name 的第一個字母轉小寫)
專有名詞
  1. Ioc(Inversion of Control)控制反轉:會將 object 存放在 spring 容器裡面
  2. DI(Dependency Injection)依賴注入:會讓我們去取得存放在 spring 容器裡面的那些 object
  3. bean:存放在 spring 容器裡的 object
  4. @Autowired:加在變數上，取得 spring 容器中的 bean
